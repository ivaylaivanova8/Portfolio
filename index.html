Проекти

Интелигентен музикален плеър с Raspberry Pi

Интерактивна система за управление на MP3 музика чрез Raspberry Pi, ESP32-S3, OLED дисплей и физически GPIO бутони.

Python Raspberry Pi ESP32-S3 GPIO OLED SSD1306 MAX98357A DAC
  • START / STOP управление
  • NEXT / PREV смяна на песни
  • Показване на песен и статус на OLED дисплей
  • Възпроизвеждане на звук през DAC аудио модул

Какво научих

Работа с Raspberry Pi, ESP32-S3, GPIO пинове, OLED дисплеи, I2C комуникация и разработване на Python приложения за управление на хардуерни устройства.

Резултат

Разработих напълно функционален MP3 плеър с физически бутони, OLED дисплей и възможност за бъдещо разширяване с Bluetooth, Wi-Fi и уеб интерфейс.
